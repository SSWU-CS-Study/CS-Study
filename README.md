## 🙏 Rule

- 스터디는 매주 일요일 오후 `7:00`에 진행
- 매회 각자 맡은 주제로 공부/발표
- 학습한 CS는 마크 다운으로 정리하여 스터디 전까지 깃헙에 업로드
- 정리 자료에는 기술 면접 질문/예상 질문이 포함되도록 작성
- 각자 공부한 것을 다른 스터디원들이 정리할 수 있도록 가르치듯이 설명 후, 스터디원들이 피드백
- 불가피한 상황으로 불참 시 최소 하루 전 언급
